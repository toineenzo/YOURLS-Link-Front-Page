# Screenshots

Drop image files into this folder so the README renders them. The README expects:

| Filename | Shows |
| --- | --- |
| `hero.png` | The public landing page (top of the README). |
| `links-tab.png` | The Links tab in the admin. |
| `image-grid.png` | The image grid on the public page. |
| `about-me.png` | The About-me block + social icons. |
| `appearance-tab.png` | The Appearance tab. |
| `quick-add.png` | The quick-add button on the YOURLS admin link table. |

Recommended dimensions: 1440×~ (will be displayed at 720 px wide).
